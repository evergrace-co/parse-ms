# parse-seconds

> Parse seoncs into an object

## Install

```
$ npm install parse-seconds
```

## Usage

```js
import parseSeconds from 'parse-seconds';

parseSeconds(1337000001);
/*
{
	days: 15,
	hours: 11,
	minutes: 23,
}
*/
```
